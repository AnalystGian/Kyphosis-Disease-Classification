# Kyphosis Disease Classification
The objective of this project is to predict whether a patient has kyphosis or not, based on given features and diagnostic measurements such as age and number of vertebrae.
Kyphosis is an abnormally excessive convex curvature of the spine.

# Learning Objectives
1. Perform data cleaning, feature engineering and visualization

2. Build, train and test a decision tree and random forest classifier models in Sklearn

3. Understand the theory and intuition behind decision trees and random forest classifier models

# Project Structure:
This project is divided into different tasks:

TASK 1: UNDERSTAND THE PROBLEM STATEMENT

TASK 2: IMPORT LIBRARIES AND DATASETS

TASK 3: PERFORM DATA VISUALIZATION

TASK 4: CREATE TESTING AND TRAINING DATASET/DATA CLEANING

TASK 5: TRAIN A LOGISTIC REGRESSION CLASSIFIER MODEL

TASK 6: EVALUATE TRAINED MODEL PERFORMANCE

TASK 7: UNDERSTAND THE THEORY AND INTUITION BEHIND DECISION TREES AND RANDOM FOREST CLASSIFIER MODELS

TASK 8: IMPROVE THE MODEL
